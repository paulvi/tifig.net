FAQ

  • Where does the name Tifig come from?
    Tifig is a Swiss German word and means "swift".
  • Is Tifig open source?
    Not yet. Tifig is part of my master thesis and I can't open source it until that project is done. However, it is my goal to make it available as open source in early 2017. Meanwhile, I plan to release a new alpha version of Tifig every two or three weeks.
  • Which versions of Swift are supported?
    Tifig requires at least Swift 3.0, which is expected to be released in late 2016 and will bring a lot of changes to the language. It is also the first release to include the Swift Package Manager, which Tifig uses to build programs.
  • Can I use Tifig for real-world software development?
    No. Tifig is at a very early stage of development and there are a lot of essential features that are still missing or incomplete. The main reason why I released an alpha version already, is because I'd like to get feedback early on. If you have any suggestions or bug reports, you can send me an email or a DM on Twitter.
