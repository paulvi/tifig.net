Download & Installation

OS X

  1. Download the current version of Tifig for OS X: Alpha Release 0.1.0 (64-Bit)
  2. Install Xcode
  3. Download & install a recent Swift 3.0 Trunk Development Snapshot

Ubuntu

  1. Download the current version of Tifig for Ubuntu: Alpha Release 0.1.0 (64-Bit)
  2. Download a recent Swift 3.0 Trunk Development Snapshot
  3. Make sure that the bin subfolder is in your $PATH environment variable. This can be done by adding something like the following line to ~/.profile: 
    export PATH=~/Downloads/swift-DEVELOPMENT-SNAPSHOT-YYYY-MM-DD-a-ubuntu15.10:$PATH

Please note that Tifig won't work with Swift 2.2 or earlier, because those versions don't contain the Swift Package Manager yet.
