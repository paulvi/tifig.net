Download & Installation

OS X

  1. Download the current version of Tifig for OS X: Alpha Release 0.2.0 (64-Bit)
    Alternatively, you can also use Homebrew to install the newest version of Tifig on your Mac: 
    brew update
brew cask install tifig
  2. Install Java 8 (Instructions for OS X)
  3. Install the newest beta of Xcode 8 (includes Swift 3.0 Preview)

Ubuntu

  1. Download the current version of Tifig for Ubuntu: Alpha Release 0.2.0 (64-Bit)
  2. Install Java 8 (Instructions for Ubuntu)
  3. Install the most recent Swift 3.0 Preview
  4. Make sure that the bin subfolder is in your $PATH environment variable. This can be done by adding something like the following line to ~/.profile: 
    export PATH=~/Downloads/swift-3.0-PREVIEW-X-ubuntu14.04/usr/bin:$PATH

Please note that Tifig won't work with Swift 2.3 or earlier, because those versions don't contain the Swift Package Manager yet.
